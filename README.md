# Yurineko PC Viewport

[English Below]

Zoom trang web đọc truyện trên Yurineko cho máy tính

## Hướng dẫn Cài đặt

### 1. Dành cho Firefox (Khuyên dùng)

Trình duyệt dùng nhân Gecko không bị siết policy Manifest V3. Cài xong là có thể chạy được

- **Bước 1:** Cài đặt tiện ích Tampermonkey.
- **Bước 2:** Cài đặt script tại đây: [Yurineko PC Viewport](https://greasyfork.org/en/scripts/584254-yurineko-pc-viewport)
- **Bước 3:** Vào trang đọc truyện, panel điều khiển nằm bên mép phải. Kéo thanh trượt để zoom.

### 2. Dành cho Chromium (Chrome, Edge, Brave, Cốc Cốc...)

Do giới hạn bảo mật từ hệ sinh thái Manifest V3 của Google, trình duyệt sẽ chặn thực thi script nếu không được cấp quyền.

- **Bước 1:** Cài đặt tiện ích Tampermonkey.
- **Bước 2:** Truy cập vào trang quản lý Tiện ích của trình duyệt (ví dụ: `chrome://extensions` hoặc `edge://extensions`).
- **Bước 3 BẮT BUỘC:** Bật công tắc **Developer mode** (Chế độ dành cho nhà phát triển) ở góc màn hình.
- _(Lưu ý: Gần đây Chrome có thêm tùy chọn "Allow User Scripts" trong cài đặt chi tiết của Tampermonkey, đảm bảo công tắc này cũng được bật)._
- **Bước 4:** Cài đặt script tại đây: [Yurineko PC Viewport](https://greasyfork.org/en/scripts/584254-yurineko-pc-viewport)
- **Bước 5:** Vào truyện và trải nghiệm.

# English Version

Optimized PC viewport for Yurineko. Bare-metal performance, strictly zero-bloatware.

## Browser-Specific Setup

### 1. For Firefox (Recommended)

Firefox's Gecko engine isn't restricted by Google's Manifest V3 policies. Plug and play.

- **Step 1:** Install the Tampermonkey add-on.
- **Step 2:** Install the script here: [Yurineko PC Viewport](https://greasyfork.org/en/scripts/584254-yurineko-pc-viewport)
- **Step 3:** Open any chapter. The zoom panel is on the right side.

### 2. For Chromium-Based (Chrome, Edge, Brave, etc.)

Due to Manifest V3 security enforcement, you must explicitly grant permission for UserScripts to execute in the DOM.

- **Step 1:** Install Tampermonkey from your Web Store.
- **Step 2:** Navigate to the extensions management page (e.g., `chrome://extensions` or `edge://extensions`).
- **Step 3 MANDATORY:** Toggle on **Developer mode** (usually located in the top-right corner).
- _(Note: Chromium updates now require toggling "Allow User Scripts" in Tampermonkey's specific extension settings as well)._
- **Step 4:** Install the script here: [Yurineko PC Viewport](https://greasyfork.org/en/scripts/584254-yurineko-pc-viewport)
- **Step 5:** Open a chapter and use the slider.
